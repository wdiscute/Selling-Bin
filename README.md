### wd's Selling bin adds a way for players to exchange their items for different currencies. 💵

\[❗\] 👷‍♀️ This mod does not include any trades by default! Scroll down or check the wiki for built-in datapacks you can use and instructions on how to make your own.

![image](https://media.forgecdn.net/attachments/description/1483881/description_5fa4cdad-86cc-4d0f-97ed-0b4c01928212.png)

Leftover from selling is stored inside the bin so nothing is lost!

# Price Tooltips

*   To view the price of an item you can simply hover the item and hold shift. 💸 This behaviour can be changed in the config accessible in-game in `Mods > wd's Selling Bin > Config`

# Mod Support & Customization (processors)

The price of an item is calculated using processors. Processors allow for nbt-based valueing, for example, if an item is damaged or has an enchantment.

## Mod Support

*   Create: Numismatics bank cards can be used to automatically transfer the sales to your bank account

*   Optional datapack for Create: Numismatics coins as currency
*   Optional datapack for Lightman's Currency's coins as currency
*   Optional datapack for Quality Food using a custom processor
*   Optional datapack for [Let's Do] Brewery beers using a custom processor
*   Optional datapack for [Let's Do] Vinery wines using a custom  processor
*   Optional datapack for Starcatcher's fishes size, weight, percentile & golden values

Got a compatibility suggestion? Let me know on discord 💸

## Custom Processors

There are a few built-in processors, and I am always happy to add more as they are requested! Reach out to me on discord if you'd like to see a built-in processor for compatibility with a specific mod. Check the wiki for the list of processors you can use directly in the datapack jsons and their properties
